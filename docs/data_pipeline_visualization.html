Gomoku VLA Data Shape

한 줄의 JSONL record는 오목 한 수를 의미한다. 현재 설계는 전략 모델이 고른 move, MuJoCo 이미지, scripted robot action, 안전 검증을 한 묶음으로 저장한다.

One Move Pipeline

한 판 전체는 여러 step의 누적이고, 각 step은 돌 하나를 놓는 record다.

state board_before 현재 15x15 grid와 current player를 읽는다.
policy strategy label CheckpointPolicy가 row, col, policy_probs, value를 만든다.
vision images top, iso, robot_full 카메라의 before/after 렌더를 저장한다.
action robot_action pick/place pose와 Cartesian action sequence를 기록한다.
check safety + result workspace, supply, legality, placement success를 검증한다.

Training Flags

training_usable 이미지 크기가 최소 224x224 이상이면 true.
phase_images_enabled phase 이미지 옵션을 켠 record인지 표시.
safety.ok pick, place, trace 검증을 모두 통과했는지 표시.
vision pending 다음 작업은 top image에서 board grid를 복원해 record에 검증 결과를 붙이는 것.
