# Study Notes

## Core Concepts

- MuJoCo: 물리 기반 로봇 시뮬레이션과 렌더링
- Gomoku policy: rule/search-based와 learning-based policy 구분
- Minimax: search-based baseline
- DQN: learning-based 또는 RL-based baseline
- VLA: vision-language-action 모델로 시각 입력과 언어/목표를 action으로 연결

## Project Framing

이 프로젝트의 핵심은 단순 pick-and-place가 아니라 오목판 상태 인식, 전략 판단, 로봇 조작을 연결하는 Gomoku-aware VLA 시스템이다.
